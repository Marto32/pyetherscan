pyetherscan
===========
An unofficial wrapper for the `Etherscan <https://etherscan.io>`_ API.

pyetherscan currently supports the following python versions:
    - ``2.7``
    - ``3.3``
    - ``3.4``
    - ``3.5``
    - ``3.5-dev``
    - ``3.6``
    - ``3.6-dev``
    - ``3.7-dev``
